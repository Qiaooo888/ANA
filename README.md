# ana包 - 简约数据分析工具

## 📊 简介
ana包是一个用R语言开发的简约数据分析工具，提供快速的数据探索和可视化功能。

## ✨ 主要功能
- `ana()` - 基础数据分析，支持自动变量选择
- `alook()` - 可视化分析，生成美观的图表
- `avar()` - 批量分析所有变量，提供完整的数据概览

# 从GitHub安装
# install.packages("devtools")
